Sign in via UMGST
Note for Web Apps:
Web browsers cannot directly read the `mc_jwt.json` file from the user's computer due to security sandboxing.

This button demonstrates how to launch the tool. To complete the login, you would typically need a companion desktop agent or ask the user to manually paste the token/code if this were a purely client-side app.
